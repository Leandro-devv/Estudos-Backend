# 🚀 Estudos de Back-End | Node.js & SQL

Bem-vindo ao meu repositório de estudos! Aqui registro minha jornada de transição da área de usinagem e mecânica de precisão para o desenvolvimento de sistemas. 

Este repositório foca no domínio do ecossistema Node.js, lógica de programação para servidores e integração com bancos de dados relacionais.

## 🛠️ Tecnologias e Ferramentas
* **Node.js** (Ambiente de execução)
* **Express** (Framework para APIs)
* **JavaScript (ES6+)**
* **MySQL** (Banco de dados)
* **Postman/Insomnia** (Teste de rotas)

## 📁 Estrutura do Repositório

* **01-fundamentos-node:** Primeiros passos com módulos internos e comandos básicos.
* **02-logica-js-backend:** Exercícios práticos de manipulação de dados (Métodos de Array como `.filter()`, `.find()` e `.map()`).
* **03-mysql-e-database:** Scripts SQL de criação de tabelas, relacionamentos e consultas (DML e DDL).
* **04-api-rest-express:** Construção de endpoints dinâmicos, tratamento de parâmetros de rota (`req.params`) e Status Codes.

## 📈 Evolução
Atualmente focado em entender a arquitetura de APIs REST e como conectar o servidor ao banco de dados para criar aplicações completas e persistentes.
